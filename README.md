# ParaMed - Mobile Medical Reference App

A mobile-first medical reference application for paramedics, designed for fast, reliable access to critical information in emergency situations.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![React](https://img.shields.io/badge/react-18.3+-61dafb)
![TailwindCSS](https://img.shields.io/badge/tailwind-4.0-38bdf8)

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 📱 Features

### Core Functionality
- **Quick Search** - Fast search across diseases, medications, symptoms, and protocols
- **Disease Database** - Comprehensive information on emergency medical conditions
- **Medication Reference** - Drug information with dosing, contraindications, and alerts
- **Symptom Guide** - Symptom assessment with possible causes and actions
- **Protocol Algorithms** - Step-by-step emergency treatment protocols

### Design Principles
- ✅ **Mobile-First** - Optimized for 320px - 448px screens
- ✅ **Dark Mode** - Eye-friendly dark interface
- ✅ **High Contrast** - Medical-grade readability

## 🎨 Technology Stack
- **React 18.3+** with TypeScript
- **React Router v7** (Data mode)
- **TailwindCSS v4**
- **Lucide React** (icons)
- **Vite** (build tool)
